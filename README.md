# Vanilla Enough Items

Vanilla Enough Items (VEI) is a Minecraft server-side plugin similar to [NEI](https://bitbucket.org/mistaqur/nei_plugins/wiki/Home), [JEI](https://www.curseforge.com/minecraft/mc-mods/jei), or [REI](https://www.curseforge.com/minecraft/mc-mods/roughly-enough-items). It allows players to type `/craft` or similar commands to open the recipe interface.
